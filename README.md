# 🧮 Simple Calculator in C

## 📖 About

This project is a **Simple Calculator** developed using the C programming language during my **first year of B.Tech (Computer Science & Engineering)**.

The calculator performs basic arithmetic operations based on the operator entered by the user. It is designed to strengthen my understanding of conditional statements (`if-else`), user input, variables, and arithmetic operations in C.

---

## ✨ Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Handles division by zero
- Displays an error message for invalid operators

---

## 🛠 Technologies Used

- C Programming
- GCC Compiler
- Visual Studio Code

## 🚀 How to Run

### Compile the program

```bash
gcc cal.c -o calculator
```

### Run the program

```bash
./calculator
```

> **For Windows (MinGW):**

```bash
calculator.exe
```
